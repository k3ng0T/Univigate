# DataHub ВУЗ-ов РК

**Версия:** 1.0.0 (First Version)  
**Статус:** Первая версия проекта

## Описание

DataHub ВУЗ-ов РК — это единая цифровая экосистема высшего образования Республики Казахстан. Платформа представляет собой централизованный агрегатор данных, обеспечивающий прозрачность, доступность и полноту информации о высшем образовании в РК.

Это не просто каталог, а интерактивная платформа, объединяющая данные, визуализацию и сервисы подачи документов для абитуриентов, студентов и академического сообщества.

## Основные возможности

### Core Features (Основной функционал)

1. **Цифровой паспорт университета**
   - Динамический таймлайн истории
   - Панель лидерства с научными индексами
   - Витрина достижений и рейтингов
   - Инфраструктурная карта

2. **Реестр образовательных программ**
   - Интеллектуальный каталог с умными фильтрами
   - Визуализация учебного плана
   - Профили преподавателей
   - Компетентностная модель

3. **Агрегатор приёмной кампании**
   - Калькулятор ЕНТ с вероятностью поступления
   - Трекер статуса заявки
   - Финансовый калькулятор
   - База грантов и скидок

4. **3D-туры и VR-интеграция**
   - 360° панорамы кампусов
   - Поддержка VR-очков (Oculus, Google Cardboard)
   - Интерактивные хотспоты
   - Режим "День из жизни"

5. **Международная академическая мобильность**
   - Интерактивный глобус с партнёрами
   - Калькулятор академической мобильности
   - Истории успеха студентов
   - Visa Support Center

6. **Аналитический движок сравнения**
   - Сравнение до 4 ВУЗов одновременно
   - Радарные диаграммы
   - Сравнение учебных планов
   - Экспорт отчётов в PDF

### Extended Features (Расширенный функционал)

- **Карьерный навигатор**: Интеграция с рынком труда, прогнозирование профессий
- **AI-консультант**: Профориентационное тестирование на базе LLM
- **Блокчейн-верификация**: Верификация достижений абитуриентов
- **Модуль общежитий**: Booking System, рейтинги инфраструктуры

## Технологический стек

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Стилизация**: Современный CSS с градиентами и анимациями
- **Шрифты**: Inter (Google Fonts)

## Установка и запуск

1. Клонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в современном браузере
3. Для локального сервера (рекомендуется):
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server)
   npx http-server
   ```
4. Откройте браузер и перейдите на `http://localhost:8000`

## Использование

### Поиск ВУЗа
- Используйте поисковую строку на главной странице
- Фильтруйте ВУЗы по категориям (Национальные, Технические, Медицинские, Экономические)

### Поиск программы
- Перейдите в раздел "Программы"
- Используйте фильтры: язык обучения, форма обучения, минимальный балл ЕНТ, максимальная стоимость

### Калькулятор ЕНТ
1. Перейдите в раздел "Поступление"
2. Введите ваши баллы по предметам
3. Нажмите "Рассчитать"
4. Система покажет доступные программы с вероятностью поступления

### Сравнение ВУЗов
1. Перейдите в раздел "Сравнение"
2. Выберите от 2 до 4 ВУЗов из выпадающих списков
3. Нажмите "Сравнить"
4. Просмотрите детальную таблицу сравнения
